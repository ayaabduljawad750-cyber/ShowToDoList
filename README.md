# 📝 Basic MEAN Stack To-Do List Display

A simple web application built using the MEAN Stack (MongoDB, Express.js, Angular, Node.js) that displays a list of to-do items.

## 🧑‍💻 Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/ayaabduljawad750-cyber/ShowToDoList
cd ShowToDoList
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
```

Run the server:

```
npm start
```

Server runs on:
http://localhost:3000

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
ng serve
```

Frontend runs on:
http://localhost:4200

---
